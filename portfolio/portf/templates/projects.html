Project 1:From Sounds to Diagnosis: Multimodal CNNs Transforming Pulmonary Healthcare

This project utilizes a respiratory audio dataset to identify pulmonary diseases such as Asthma, Pneumonia, and Bronchiectasis by extracting features from audio data and training a convolutional neural network (CNN) model, which allows for accurate disease prediction from new test audio data.

Project 2:Multi-Step Verification System for Enhanced User Security

This project enhances online security by introducing a novel three-step verification process for user registration and login, involving an 8-character password, a chosen color sequence, and a specific area within a selected picture, all securely stored using a MySQL database to prevent unauthorized access.
